IndoLaw Scraper

Indonesian Government Document Scraper

Automated Python-based scraper for Indonesia’s JDIH portals — collecting open legal data for transparency and research.

← Back to Technology & Innovation

Repository Structure

Each scraper module targets a specific JDIH portal from Indonesian government institutions.

Coordinating Ministry for Maritime and Investment Affairs (Kemenko Marves)
Ministry of Public Works and Housing (PUPR)
Other Institutions (Still Progress)

How to Use IndoLaw Scraper

Install the package and run scrapers from Python CLI to extract open legal documents from Indonesian government websites.

Install via PyPI

pip install indolaw-scraper

Scrape an Audit Board of the Republic of Indonesia (BPK) Document

python -m indolaw_scraper.cli --source bpk --url "https://peraturan.bpk.go.id/Details/227490/peraturan-kpk-no-5-tahun-2018"

TNI Document Scraper

The TNI Scraper module collects Peraturan Panglima TNI (Commander-in-Chief Regulations) from the official JDIH TNI portal. Unlike most metadata scrapers, this module also parses the PDF files to extract the first 1,000 characters of each regulation — enabling contextual previews for analysis and indexing.

Command-Line Usage

python -m indolaw_scraper.cli --source tni --url "https://jdih.tni.mil.id/peraturan-panglima-tni"

Notes:
• The scraper retrieves data from the /api/produk-uud/<page> endpoint of JDIH TNI.
• Each PDF is processed with PyMuPDF (fitz) to extract a short preview text (~1000 characters).
• The output can be exported as a pandas DataFrame or JSON for integration with analysis pipelines or web dashboards.
